Considere uma classe Pilotos com um atributo privado String nome e outro atributo privado pistaCorrida do tipo Pista; 
Considere o uso do método hashCode() para imprimir o ID da pista;

Para a aplicação, desenvolva um projeto Singleton que permita o cadastro de n corredores para disputarem uma corrida em uma pista que se chama “Autodromo Internacional de Goiania” (pode usar o construtor da Pista para inicializar esse nome).
